## How to Use

### With Mouse
1. **Click** any element
2. **Hear** it read aloud
3. **Navigate** by clicking

### With Keyboard
1. **Press Tab** to move between elements
2. **Hear** each element as you focus
3. **Press Enter/Space** to activate

### With Voice Navigation
1. **Say** voice command
2. **Hear** confirmation
3. **Navigate** hands-free

## Troubleshooting

| Problem | Solution |
|---------|----------|
| No sound | Check volume, enable screen reader |
| Wrong voice | Change in OS speech settings |
| Choppy speech | Refresh page, close extra tabs |
| Not all elements read | Ensure elements have labels/ARIA |
