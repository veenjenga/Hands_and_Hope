## ğŸ”§ Troubleshooting

### Voice Commands Not Working?

#### Check the Status Indicator:
- ğŸ¤ **Green "Listening..."**: System is active and ready
- ğŸ”‡ **Gray "Not Listening"**: Voice navigation is off
- âš ï¸ **Yellow Warning**: Microphone permission needed

#### Common Issues:

**1. Microphone Permission Denied**
- Click the ğŸ”’ lock icon in the address bar
- Change microphone permission to "Allow"
- Refresh the page and try again

**2. "Not Listening" Status**
- Toggle voice navigation OFF then ON again
- Check if another app is using your microphone
- Make sure you're on HTTPS (secure site)

**3. Commands Not Recognized**
- Speak more clearly and slowly
- Reduce background noise
- Try alternative command phrases
- Check "Last command" to see what was heard

**4. No Microphone Found**
- Connect a microphone or headset
- Check system sound settings
- Restart your browser
