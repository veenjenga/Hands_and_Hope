## â“ Common Questions

**Q: Where do I start?**
A: Read [ACCESSIBILITY_FEATURES.md](./ACCESSIBILITY_FEATURES.md) first!

**Q: Voice navigation not working?**
A: See [FIGMA_IFRAME_VOICE_NAV_FIX.md](./FIGMA_IFRAME_VOICE_NAV_FIX.md) - likely an iframe issue.

**Q: What's the difference between screen reader and voice navigation?**
A: Screen reader reads TO you (output). Voice navigation listens to YOU (input).

**Q: Can I use multiple features at once?**
A: Yes! All features work together seamlessly.

**Q: Which browser is best?**
A: Chrome or Edge for best experience.

**Q: Do these features work on mobile?**
A: Yes! iOS Safari and Android Chrome fully supported.
